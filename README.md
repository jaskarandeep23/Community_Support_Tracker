# Community Support Tracker

## Description
Community Support Tracker is a web application that allows users to submit and track donations. It includes form validation and automated tests to ensure data integrity.

## Features
- Donation form with validation
- Create donation objects
- Form submission handling
- Unit testing using Jest & JSDOM
- CI/CD testing with GitHub Actions
